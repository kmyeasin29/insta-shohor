Javascript is a single threaded high level programming language, It is also known as scripted language. It is designed based on the features of the browser. It does not have to be compiled like C++ language. When you using high level language you don't need to think about how the machine works, how it compiles, how it runs, how much space it takes up memory. It will do the rest instead of you.

JavaScript is usually based on the event loop. Executes event loop code, collects events and performs all tasks through processing. The JavaScript engine never executes multiple codes at once, it executes one code after another according to the serial. But in the meantime, if time is set to run a code, JavaScript does not run it immediately and notes it. And continues that after the fix time code. This process is known as JS event loop.
